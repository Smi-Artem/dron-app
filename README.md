Краткая аннотация:

SmArtSeeker — бюджетное решение для автоматизации поисково-спасательных операций в лесных массивах. Комплекс включает беспилотный летательный аппарат с бортовой нейросетью YOLOv8n, радиоканал передачи данных и приложение наземной станции оператора. Проект направлен на снижение нагрузки на спасательные группы и повышение эффективности поиска за счёт автоматического обнаружения людей.

Приложение для управления поисково-спасательным дроном. Отображает местоположение дрона на карте, принимает и визуализирует данные с камеры, сохраняет координаты обнаруженных объектов. Построено на PyQt.
